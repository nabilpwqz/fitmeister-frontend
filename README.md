# 🏋️ FITMEISTER — Fitness & Gym Management Platform (Frontend)

## 🚀 Project Overview
IronPulse is a modern fitness platform where users can discover, book, and manage fitness classes while engaging in a community-driven forum. This frontend delivers a seamless, responsive, and interactive user experience across all roles — Users, Trainers, and Admins.

## 🎯 Purpose
The frontend is responsible for:
- Delivering an intuitive UI/UX
- Handling client-side routing
- Managing authentication states
- Communicating with backend APIs
- Rendering role-based dashboards dynamically

---

## ⚙️ Tech Stack
- React.js
- React Router DOM
- Tailwind CSS / DaisyUI
- Axios
- Framer Motion
- React Hook Form
- Stripe.js
- Firebase / Better Auth (for authentication)

---

## 🔑 Key Features

### 🔓 Public Features
- Browse all fitness classes
- Search & filter classes by category/name
- View community forum previews
- Authentication (Login/Register with Google support)

### 🔐 Private Features
- Book classes with Stripe integration
- Add/remove favorite classes
- View detailed class information
- Participate in forum (like, comment, reply)

### 👤 User Dashboard
- View booked classes
- Track favorite classes
- Apply to become a trainer

### 🧑‍🏫 Trainer Dashboard
- Create & manage classes
- View enrolled students
- Post in community forum

### 🛡️ Admin Dashboard
- Manage users (block/unblock/promote)
- Approve/reject trainers & classes
- Monitor transactions
- Moderate forum posts

---

## 🔐 Security
- Environment variables used for API URLs and keys
- Protected routes with role-based access control
- JWT handled via HTTP-only cookies
